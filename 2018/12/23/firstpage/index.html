在git的方面，我没有安装文章的方式安装。而是将自己的repository进行clone到本地。然后将hexo文件内的全部文件都移动到clone后的文件。然后执行了一次

  • git add .
  • git commit -m ‘init’
  • git push origin master

命令之后，然后才安装部署工具。

但是，没有设置config.yml 文末最后的deploy，结果没有任何反应。
