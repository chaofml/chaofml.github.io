开始使用hexo

在github上使用hexo

安装过程，参照文章，写的比较详细。安装比较顺利。之前在服务器也安装过hexo，不过从来没有更新过东西。

至此，博客已经安装成功

整个安装过程比较简单，可能是因为平时也使用过了hexo已经git工具、npm工具，所以在理解文章的内容方便，没有任何困难。我安装，并不是严格的安装文章的方式来弄的。下面是我的安装：

在git的方面，我没有安装文章的方式安装。而是将自己的repository进行clone到本地。然后将hexo文件内的全部文件都移动到clone后的文件。然后执行了一次

  • git add .
  • git commit -m ‘init’
  • git push origin master
    命令之后，然后才安装部署工具。

但是，没有设置config.yml 文末最后的deploy，结果没有任何反应。
然后仔细看文章后，加上了deploy的相关设置，成功部署。

部署的实质

部署的实质其实利用hexo这个博客工具，产生静态文件，放置在public文件夹下面，并利用deploy工具push到git上面。
