mysql查看连接数

查看mysql的当前连接数，查看配置等。

  • 1、查看当前所有连接的详细资料:

　　./mysqladmin -uadmin -p -h10.140.1.1 processlist

  • 2、只查看当前连接数(Threads就是连接数.):

　　./mysqladmin -uadmin -p -h10.140.1.1 status

如果使用sql语句，如下

show full PROCESSLIST;

show STATUS like 'Threads%'，Threads_connected即为连接数。

  • 3、mysql中最大连接数：

    max_connections=100(默认为100)

使用navicat等查看：


show status like '%connect%';
-- 结果解析
-- Connections： 连接的数量，不论成功与否。貌似连接、断开都会加1。累计
-- Max_used_connections  最大连接数   注意，最大
-- Threads_connected   当前连接数  跟show PROCESSLIST; 不一致


-- 配置项
show variables like '%connect%';
-- 结果解析
-- max_connections 最大连接数


-- 查看当前的连接
show PROCESSLIST;
