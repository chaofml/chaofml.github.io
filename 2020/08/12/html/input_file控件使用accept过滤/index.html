二、限制规则：

在文件上传控件的accept属性中，接受以下两种格式的限制符：

  • 文件类型，但不限制具体扩展名，如：image/*，就只要是图片就行。
  • 文件类型+具体的文件扩展类型，如：image/jpeg，使用的是MIME_TYPE类型
  • 多个MIME_TYPE类型之间用逗号分隔。
