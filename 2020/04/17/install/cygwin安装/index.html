安装过程cygwin

第一步：下载安装新版Cygwin软件。该软件的官方网站：http://cygwin.com/

  1. 要编译C++或者C程序的话，再安装时在选择组件这一步需要选中Devel 选项下的gcc-core,gcc-g++,make 以及Lib选项卡下的glib2,glib2-devel,glib2-runtime。

  2. 为了提高网络下载速度，建议下载网址选用网易镜像站点， 在软件界面，需要填url的地方，填下面的地址。
