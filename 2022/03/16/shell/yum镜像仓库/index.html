使用

配置说明

name=Description#一个描述，随意。
baseurl=#设置资源库的地址，可以写阿里云也可以是自己的yum
    ftp://
    http://
    file:///
enabled={1|0}#enabled=1开启本地更新模式
gpgcheck={1|0}# gpgcheck=1表示检查；可以不检查gpgcheck=0
gpgkey=#检查的key；如果上面不检查这一行可以不写。

使用光盘镜像

假设所有的内容都挂载到了/mnt/ 目录下


cd /etc/yum.repos.d/
# 可以先备份或删除
rm -rf *

# 更改配置
cat centos7.repo <<EOL
[centos7]
name=centos7
baseurl=file:///mnt
enabled=1
gpgcheck=0
#gpgkey=/key/path
EOL

# 更新
yum clean all
yum list all
# 测试
yum install  mcelog
