压缩、解压

发表于 2022-09-14 更新于 2023-01-20 分类于 shell

Linux环境下，压缩与解压的命令。

gzip

# 压缩   并删除原文件
gzip test3

# 查看信息 
gzip -l test3.gz 

# 解压
gunzip test3.gz 
gzip -dv test3.gz  #-d 解压  -v 显示信息

compress

# 安装
apt install ncompress
# 压缩    并删除原文件
compress test3
# 解压
uncompress test3.Z

zip

apt install zip
# 压缩
zip test3.zip test3
# 解压
unzip test3.zip

xz

tar xvJf  ***.tar.xz

# 两部方式
xz  -d   linux-3.8.4.tar.xz  # 解压出 linux-3.8.4.tar
tar -xvf  linux-3.8.4.tar

bz

# 解压
bzip2 -d FileName.bz
bunzip2 FileName.bz
tar jxvf FileName.tar.bz

bz2


解压1：bzip2 -d FileName.bz2

解压2：bunzip2 FileName.bz2

压缩： bzip2 -z FileName.tar.bz2
解压：tar jxvf FileName.tar.bz2

压缩：tar jcvf FileName.tar.bz2 DirName

tar

参数

-c: 建立压缩档案
-x：解压
-t：查看内容
-r：向压缩归档文件末尾追加文件
-u：更新原压缩包中的文件

下面的参数是根据需要在压缩或解压档案时可选的。
-z：有gzip属性的
-j：有bz2属性的
-Z：有compress属性的
-v：显示所有过程
-O：将文件解开到标准输出

下面的参数-f是必须的
-f: 使用档案名字，切记，这个参数是最后一个参数，后面只能接档案名。

# 这条命令是将所有.txt的文件增加到all.tar的包里面去。-r是表示增加文件的意思。
tar -rf all.tar *.txt

# 这条命令是列出all.tar包中所有文件，-t是列出文件的意思
tar -tf all.tar


#这条命令是将所有的文件打成一个包。-c是表示产生新的包，-f指定包的文件名。
tar -cvf dzz.tar * 

#将/home/stud1目录做归档压缩，压缩后生成wang.tar.gz文件，
#并将此文件保存到/home目录下，实现此任务的tar命令格式,-z：压缩
tar zcvf /home/wang.tar.gz /home/stud1

# 将目录里所有txt文件打包成dzz.tar后，并且将其用compress压缩，生成一个umcompress压缩过的包，命名为dzz.tar.Z
tar -cZf dzz.tar.Z *.txt
